选股条件
1. 趋势确认（右侧）：股价已从近期低点反弹5%-10%以上，或站上某条中期均线（如20日线）
2. 等待金叉：3日均线从下方向上穿越16日均线
3. 成交量配合：金叉当日或前几日有温和放量，说明有资金进场
可用选股公式思路：
MA3 > MA16 AND REF(MA3,1) < REF(MA16,1)  // 今日金叉
AND C > REF(LLV(L,20),5) * 1.05           // 股价已从20日低点反弹5%以上
AND VOL > MA(VOL,5)                        // 成交量放大

---
买入时机
- 信号日：3日线上穿16日线当天收盘确认
- 买点：次日开盘价附近，或回踩16日均线不破时

---
仓位管理
情况情况	建议仓位
大盘弱势/震荡市	1/3仓试探
大盘企稳/温和上涨	半仓
大盘强势+个股形态好	6-7成仓
全仓	不建议，除非极高确定性
分批建仓：首次买入后，若股价沿均线上行、不破16日线，可在回踩时加仓。

---
卖出条件
1. 止损：跌破16日均线且3日内无法收回，或亏损达5%-7%
2. 止盈：
  - 短线：盈利10%-15%可减半仓
  - 中线：3日线下穿16日线（死叉）清仓
  - 或股价远离均线、乖离率过大时分批止盈

---
简化操作流程
1. 每日收盘后筛选出现金叉的股票
2. 剔除大盘弱势期间的信号
3. 次日择机买入，设好止损位（16日线下方2%-3%）
4. 持有期间：不破16日线就拿着
5. 出现死叉或破位止损离场

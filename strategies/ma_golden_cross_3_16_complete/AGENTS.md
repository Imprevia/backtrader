# ma_golden_cross_3_16_complete - 均线金叉3穿25策略（收益优化版）

**策略目录**: `strategies/ma_golden_cross_3_16_complete/`

**继承**: `AShareStrategy` (backtest_engine.py)

---

## 概述

完整版右侧均线金叉策略，采用**3日均线与25日均线金叉死叉**作为核心信号，配合ATR动态止盈体系。包含选股质量评分、死叉移除候选、完整止损止盈体系。

**核心特点**：
- 3/25日均线组合（稳定信号，过滤假突破）
- ATR动态止盈（4.5×ATR完全止盈）
- 趋势过滤（20日线向上）
- 候选股死叉移除（防止追高）
- 分批止盈（2.5×ATR减半仓 + 4.5×ATR全清）

## 选股流程

### 选股条件（同时满足）

| 条件 | 说明 |
|------|------|
| **1. RSI过滤** | RSI>30（动态计算） |
| **2. 趋势过滤** | 20日线连续5日向上 |
| **3. 趋势确认** | 股价从近期低点反弹4%-25% |
| **4. 金叉即将发生** | 3日均线在25日均线附近（差距<5%） |
| **5. 成交量配合** | 金叉前3日内有放量（超过20日均量1.3倍） |

### 候选股死叉移除

- 候选股加入后，每日扫描检测死叉
- 如果3日均线下穿25日均线，立即从候选名单移除
- 防止追高买入已形成死叉的股票

### 候选股超时清理

- 候选股等待金叉最多20个交易日
- 超过20日仍未金叉则从候选列表移除
- 已移除股票加入黑名单，避免重复筛选

### 候选股质量评分

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 金叉临近度 | 30% | 快线越接近慢线分数越高 |
| 放量强度 | 30% | 放量倍数越高分数越高 |
| 趋势强度 | 20% | 股价在20日线上方幅度 |
| 相对强弱 | 20% | 从低点反弹幅度适中加分 |

## 买卖信号

### 买入阶段

```
T日收盘扫描：
  RSI过滤（RSI>30）
  + 趋势过滤（20日线向上5日）
  + 反弹4%-25%
  + 快线接近慢线±5%以内
  + 放量1.3倍+
  → 加入候选列表
  → 检测死叉则移除

T+1日开盘：
  金叉确认（快线已穿越慢线）+ 站稳均线
  → 以开盘价买入
```

### 卖出信号（按优先级）

| 优先级 | 触发条件 | 操作 | 卖出原因 |
|--------|----------|------|----------|
| 1 | 死叉 | 清仓 | 3日均线下穿25日均线 |
| 2 | 跌破25日均线且3日无法收回 | 清仓 | 均线止损 |
| 2 | 亏损达5% | 清仓 | 百分比止损 |
| 3 | 盈利达到2.5×ATR且未减仓 | 减半仓 | ATR动态止盈（第一目标） |
| 3 | 盈利达到4.5×ATR | 清仓 | ATR动态止盈（第二目标） |
| 4 | 乖离率>15%且已减仓 | 清仓 | 乖离率过大分批止盈 |

## ATR动态止盈体系

### 原理

使用ATR（Average True Range）衡量股价波动率，动态调整止盈点位：
- **ATR**: 14日平均真实波幅
- **2.5×ATR 减半仓**: 当盈利达到2.5倍ATR时，卖出半仓锁定利润
- **4.5×ATR 完全止盈**: 当盈利达到4.5倍ATR时，清仓离场

### 计算示例

假设：
- 买入价：10.00元
- ATR值：0.30元
- 2.5×ATR目标：10.00 + 2.5×0.30 = 10.75元（盈利7.5%）
- 4.5×ATR目标：10.00 + 4.5×0.30 = 11.35元（盈利13.5%）
